## Overview

GMGN App is a cryptocurrency copy trading platform built with Flutter, featuring:
- Simulated deposit functionality
- Token trading with K-line charts
- Copy trading with top traders
- Real-time smart money order flow

**Live Demo:** https://fbcd1012.github.io/gmgn-app/
